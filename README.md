# X浏览器拦截规则

[English Version](README_EN.md)


这是由[X浏览器官](https://www.xbext.com)方维护的拦截规则，其主要特色如下。



- 基于X浏览器自己的一些规则语法。
- 速度快，占用极少资源。
- 主要针对移网站的恶意广告进行拦截。

## 关于


我们并不鼓励拦截掉所有的广告, 对于可接受的非恶意广告我们仍希望广大网友允许他们的存在，毕竟这是站长赖以生存的方式。
